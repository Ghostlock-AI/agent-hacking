Web-Facing Agent Demo

This container runs a FastAPI-based chat endpoint on port 8000.

Warning: This example is for educational use only. Do not expose it to the public internet or run the insecure compose profile outside an isolated environment.

Try running: docker compose up web-agent and then POST to http://localhost:8000/chat.

Insecure profile (for demonstration only): docker compose --profile insecure up web-agent-insecure
